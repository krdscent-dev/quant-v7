# 系统设计说明

## 1. 目标

构建一个围绕 A 股主题研究的量化系统，用于统一整理：

- 主题股票池
- τ因子
- 华为超节点主题暴露
- 国产替代主题暴露
- 综合评分
- 简单回测

## 2. 处理流程

1. 从外部数据源或本地文件读取原始数据
2. 清洗并写入 `data/processed/`
3. 计算主题因子
4. 生成股票池与综合评分
5. 执行简单回测
6. 将结果输出到 `reports/`

## 3. 模块职责

- `factors/`
  - 定义各主题因子的输入、输出和计算接口
- `strategy/`
  - 管理股票池构建、权重配置和评分逻辑
- `backtest/`
  - 负责验证策略信号和基础绩效统计
- `config/`
  - 保存主题名单、权重、过滤规则等配置
- `scripts/`
  - 承载批处理入口，后续用于自动化执行

## 4. 当前约束

- 目前只搭建骨架，不绑定具体数据源
- 所有核心 Python 文件保留清晰函数签名和类型注解
- 后续实现应尽量保持模块边界稳定，避免频繁改动接口
