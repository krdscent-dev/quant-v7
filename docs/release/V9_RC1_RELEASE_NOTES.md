# V9 RC1 Release Notes

## 新增模块汇总

- `src/quality`
- `src/agent_workflow`
- `src/knowledge_base`
- `src/backtest`

## 主要能力

- 统一质量门禁与 RC1 Ready 判断
- 研究工作流可编排、可汇总
- 历史研究记录可沉淀与查询
- 回测、组合、仓位、风险与调仓链路可串联

## 已知问题

- 回测数据仍为研究验证用样本
- 质量门禁不替代代码审查
- 部分结果依赖内存缓存，适合 RC1，不适合长期持久化

## 升级说明

- 不新增大型业务模块
- 不改变核心研究逻辑
- 以稳定性和可交付性为优先
- RC1 只关注集成连通性与基本质量控制
