# A股 AI算力 / 华为超节点 / τ因子 / 国产替代 V7 量化研究系统

本仓库用于搭建一个面向 A 股主题研究的量化分析框架，覆盖以下方向：

- AI算力产业链
- 华为超节点相关主题
- τ因子研究
- 国产替代主题筛选

目标不是一次性给出最终可交易模型，而是先把研究系统的结构、接口和产出路径搭好，方便后续逐步填充数据、因子、打分模型和回测逻辑。

## 目录结构

- `strategy/`：股票池构建与综合评分模型
- `factors/`：主题因子与特征计算
- `backtest/`：简单回测与结果验证
- `data/`：原始数据与处理后数据
- `data/raw/`：原始输入数据
- `data/processed/`：清洗和特征化后的数据
- `reports/`：研究报告、回测结果、图表导出
- `docs/`：系统设计、研究口径、后续约定
- `config/`：研究参数、观察名单、环境配置
- `scripts/`：批处理脚本和任务入口

## 使用约定

- 先在 `data/raw/` 保留原始输入
- 处理结果输出到 `data/processed/`
- 研究结论和回测结果输出到 `reports/`
- 配置优先写入 `config/`
- 核心逻辑优先放在 `factors/`、`strategy/` 和 `backtest/`
